# OLX Sales-Intelligence CLI

This repository contains the `olx-pp-cli`, a reverse-engineered CLI and MCP server for the OLX.pl job-listings API.

## Directory Structure

- `src/` - Contains the Go source code for the CLI and MCP server, generated by [CLI Printing Press](https://github.com/mvanhorn/cli-printing-press) and customized manually. Check out [`src/README.md`](src/README.md) for more details.
- `docs/` - Contains project documentation, research notes, and build logs (e.g., [`docs/build-log.md`](docs/build-log.md)).
- `data/` - Directory for local data storage (like SQLite databases) and data exports.

## Building the Project

To build the CLI and MCP server binaries, run the following commands:

```bash
cd src
make build-all
```

The compiled binaries will be available in the `src/bin/` directory.

## Running Tests

To run the tests for the project:

```bash
cd src
make test
```
